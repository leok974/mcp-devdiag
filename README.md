# mcp-devdiag (portable export)

Model Context Protocol server exposing tools to read backend/frontend logs, env, CORS state, and network summaries.

## Run (stdio)
```bash
mcp-devdiag --stdio
```

## Files used (workspace root)
- `.tasteos_logs/backend.log`
- `.tasteos_logs/frontend.log`
- `.tasteos_logs/network.jsonl`
- `.tasteos_logs/env.json`
